Lumina translates dApp interfaces without corrupting the code-shaped values that make Web3 applications work. It protects variables, template expressions, tags, URLs, and token tickers before translation, restores them afterward, and rejects output that cannot be proven structurally safe.

## What Lumina solves

Generic translation can rename `ETH`, alter `{amount}`, drop `{{wallet}}`, or damage JSX and URLs. Lumina isolates these tokens, adds Web3 glossary context, validates every model response, and only then returns or persists the translation.

It also converts raw OKX OnchainOS and EVM failure codes into localized, actionable guidance, so users receive an explanation instead of an opaque execution error.
